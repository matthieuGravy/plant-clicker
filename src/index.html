Rules

Grow the plant

The goal of the game is to grow a plant by clicking on the screen.

Each plant is unique. Start new games to see what kind of plant you will get.

Tokens

Each time you click on the screen, you will gain some tokens. These tokens can be used to purchase upgrades.

Upgrades

There are 3 upgrades you can buy:

  • Auto-clicker: automatically clicks on the screen every second.
  • Multiplier: increase the number of tokens earned.
  • Booster: increase the number of token earned by 3 for 15 seconds.

Save

Your progress is saved automatically. Go to the Stats tab to start a new game.
